# 华师匣子（ccnubox）应用信息

## 简介
【华师匣子】简称【匣子】，是华中师范大学木犀团队打造的官方校园应用，面向华师在校学生，提供一站式校园生活服务。

## 核心能力（常用功能）
- 课表：随身课表查询与管理
- 成绩：成绩查询
- 校园卡：校园卡相关功能（如余额/消费等，具体以产品实际为准）
- 新闻公告：校内资讯与公告
- 空教室：空教室查询
- 图书馆：图书馆相关管理能力

## 产品定位与卖点
- 面向华师学子量身定制
- 将多项校务与校园生活能力整合到一个应用
- 目标是让校园日常事务“化繁为简”

## 宣传文案（可用于介绍/回答“这是什么”）
