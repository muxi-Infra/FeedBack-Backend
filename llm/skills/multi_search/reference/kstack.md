# 木犀课栈（kstack）应用信息

## 简介
【木犀课栈】是一款课程评价类小程序，面向在校学生提供课程信息查询与评价参考，帮助同学更高效地了解课程体验并做选课决策。

## 核心能力（常用功能）
- 课程评价：浏览、发布课程评价与建议（以产品实际功能为准）
- 课程信息：按课程等维度查询课程相关信息（以产品实际功能为准）
- 评价参考：通过历史评价了解课程难度、作业/考试形式、给分情况等常见关注点（以用户评价为准）

## 产品定位与卖点
- 聚焦“选课/修课体验”信息不对称问题
- 以真实用户评价为主要参考信息来源
- 帮助学生在选课前快速了解课程口碑与注意事项

## 使用场景
- 选课前：想了解某门课/某位老师的作业量、考试形式、课堂风格等
- 修课中：想参考他人经验，优化学习方法或避坑
- 修课后：沉淀自己的课程体验，为他人提供参考

## 宣传文案（可用于介绍/回答“这是什么”）
应用详情：【木犀课栈】——你的选课评价与课程经验分享小程序。它聚合课程评价与课程信息，帮助你在选课前快速了解课程口碑与常见注意事项；在修课中参考同学经验更高效地学习；修课后也能分享真实体验，给后来者一盏灯。
